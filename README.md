# AQUARA — лендинг доставки воды

Концептуальный (демо) лендинг сервиса подписки на питьевую воду глубокой очистки.

🔗 **Демо:** https://andreyqqqqq.github.io/aqua-landing/

## Задача

Создать одностраничный сайт, который превращает абстрактное обещание «чистая вода» в измеримый факт — с цифрами, лабораторным контролем и понятным путём воды от источника до стакана.

## Решение

* Уникальная визуальная концепция под тему чистоты: холодная «ледяная» палитра (белый лёд + глубокий петроль + акцент цвета аквы), фирменная типографика (Unbounded + Manrope)
* Авторский элемент «живая капля» в hero — CSS-фигура с бликами и всплывающими пузырьками, внутри которой счётчик TDS анимированно падает с 248 до 12 ppm, визуализируя очистку воды в реальном времени
* Секция «Путь капли» — 7 ступеней фильтрации в реальной технологической последовательности, где нумерация несёт смысл, а не украшает
* Полная структура одностраничника: hero, полоса ключевых цифр, ступени очистки, тарифы с выделенным планом, цитата команды, финальный CTA
* Плавное появление секций при скролле через IntersectionObserver
* Адаптивная вёрстка под все устройства
* Учтена доступность: видимый фокус для клавиатурной навигации, отключение анимаций при `prefers-reduced-motion` (счётчик сразу показывает финальное значение)
